# no_duplicates_advanced_mystdclb
This removes any duplicates from the scrapped json file its an advanced version because company was finding it difficult to remove same job posting but from different career portals  this code checks the description with threshold of(0.85) you can change it as per your needs. If it matches it removes the duplicate job posting. 

And it also takes files saved in jupyter and uploads the updated file in jupyter as per the name provided
